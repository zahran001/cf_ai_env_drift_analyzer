# Tests

## Locations
- Unit tests live under `src/**/__tests__`.

## Run All Tests
```bash
npm test
```

## Run a Targeted Test File
```bash
npm test -- src/analysis/__tests__/cacheUtils.test.ts
```

## Run a Targeted Folder
```bash
npm test -- src/analysis/__tests__
```

## Watch Mode
```bash
npm run test:watch
```

## Coverage
```bash
npm run test:coverage
```
